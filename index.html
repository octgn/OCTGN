[ok-tuh-gon] is a free virtual table top with an integrated player lobby and chat system that supports custom card and board games.

Join or host a game outside of a hobby shop. allows you to connect to other players around the world.

Interact with the growing community. is open source and frequently updated.

Numerous forums and users dedicate their time to creating and updating rule sets to add to the list of supported games.

Overview

Follow this short tutorial to get your copy of . You will be guided through each step from downloading to eventually playing. Begin your short journey by opening the next tab.

Setup and Create an Account

Download the program by clicking on the version number in the header.

Install the program from the executable and move the game shortcut into your StartUp folder.

Open . Click Octgn in the top left and choose Options then Set Data Directory. Choose a folder somewhere on your computer that has at least 5GB of free space. This is where will store your games.

Click the Register button if you do not have an account.

Fill in your email address. Your usename is how others will see you. Make sure you remember your password!

You can now access the player server. Log in with your username and password. Put a check in the Save Password box.

Obtaining Games

You need to tell how to play your game then give it cards and maybe some extra files, like, tokens before you can start making a deck and playing.

The game rules are stored in the o8g file.
The game sets/cards are stored in the o8s file.
If needed, patches are released in o8p files.
The extra cards, like tokens, are stored in o8s files.

Do a search for OCTGN games to find some files. Download the o8g and o8s files.

Log into the client. Open the Games tab and choose Install Game. Choose the o8g file that you downloaded.

Once again in the Games tab; click on your newly installed game and the list is blank! You need to install some sets. Click Install Set and choose the o8s file(s) that you downloaded.

Voice Chat

Voice chat is more convenient than typing everything you want to tell your opponent. You will also not be teased for not knowing the difference between your and you're. Skype, teamspeak and Ventrillo are common choices.

A voice chat server has been set up by Divided by Zero for users through the Mumble client. Use the button below to download the client for your computer.

Install Mumble and finish its audio wizard with your microphone or headset. Choose Server from the top menu then click Connect (CTRL+O). Fill out the server iformation. The address is mumble.dbzer0.com and the port is 63836

Drag your username into the channel you want to join.

Joining or Hosting a Game

Within open the host/join tab and choose Join to refresh the current games. By default only the games that you have installed are shown. Choose a game and meet the opponent! The lobby host will start the game when she or he is ready.

Is no one hosting a game you want to play? Host it yourself and have others join.

After others join you press Start to start the game.

The wiki is community edited and will always have the most up-to-date information.

Log into the Freenode webchat to talk with the developers and other people for help that you won't get by asking questions in the game lobby.
